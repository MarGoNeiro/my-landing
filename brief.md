## 1. Цель и аудитория

### Цель сайта

**Первичная**: Конвертировать посетителя в заявку на бесплатную консультацию.  
**Вторичная**: Установить доверие и экспертность до первого контакта, чтобы разговор начинался с "хочу работать с вами", а не "расскажите кто вы".

Не галерея работ. Не визитка. **Продающая страница**, где каждый блок устраняет конкретное возражение клиента.

### Целевая аудитория

#### Сегмент A — Владелец малого бизнеса (основной)
- Компания 2–30 человек, оборот 3–50 млн ₽/год
- Проблема: старый сайт не приносит заявки / нет сайта вообще / хочет автоматизировать рутину
- Боли: боится долго, дорого, непонятно; обжигался на фрилансерах/агентствах; не знает что такое ИИ-агент
- Принимает решение сам, ориентируется на доверие и конкретные примеры
- Устройство: ~60% мобильный, ~40% десктоп

#### Сегмент B — Стартап-фаундер / продакт
- Нужен MVP веб-приложения или AI-инструмент для продукта
- Технически подкован, оценивает стек и подход
- Ценит скорость и отсутствие посредников
- Сравнивает нескольких подрядчиков одновременно

#### Сегмент C — Маркетолог / операционный директор
- Ищет подрядчика по заданию руководства
- Нужны: кейсы, социальное доказательство, понятный процесс
- Критерии выбора: надёжность + портфолио + коммуникация

### Чего НЕ делает сайт
- Не объясняет что такое программирование в целом
- Не пытается понравиться всем — лучше отпугнуть нецелевых
- Не ставит целью "показать крутой дизайн" ради самого дизайна

## 2. Позиционирование и оффер

### Ключевое отличие
Один специалист, который закрывает три слоя цифрового продукта — сайт, приложение, ИИ-слой. Нет координационных потерь между подрядчиками, нет переводов между командами.

### Главный оффер (выбрать один вариант при наполнении)

**Вариант A** — широкая аудитория, акцент на автоматизацию:
> «Создаю сайты, приложения и ИИ-агентов — чтобы ваш бизнес работал, пока вы занимаетесь главным»

**Вариант B** — фаундеры, акцент на скорость:
> «Ваша идея заслуживает большего, чем шаблон. Запущу продукт, который реально работает»

**Вариант C** — AI-forward, дифференциация:
> «Большинство разработчиков создают инструменты. Я создаю то, что думает»

**Рекомендация**: Вариант A — самый широкий охват, легче понять владельцу бизнеса. Вариант C — если хочешь позиционировать AI как главный сервис.

### Subheadline (под главным заголовком)
> «Сайты, приложения и ИИ-агенты под ключ — в одних руках, без агентских наценок и без потерь при передаче задач»

### Антипозиционирование (чем НЕ являешься)
- Не агентство с командой в 20 человек
- Не фрилансер с шаблонами на Tilda без кода
- Не просто разработчик без понимания бизнеса

## 4. Контент секций

---

### 00 · Навигация

**Тип**: Sticky, прижата к верху, видна всегда при скролле.

**Левая часть**: Логотип или имя (текстовый логотип, моноширинный шрифт)  
**Правая часть**: Якорные ссылки + кнопка CTA

**Ссылки**:
```
Услуги   Работы   Обо мне   FAQ
                              [Обсудить проект →]  ← кнопка, всегда видна
```

**Поведение**:
- При скролле вниз: фон nav появляется (blur + semi-transparent)
- На мобильном: hamburger меню, полноэкранный overlay
- Active-состояние ссылки подсвечивается при нахождении в секции (IntersectionObserver)
- Кнопка в nav: filled accent, hover — немного светлее

**Визуальный хук — AI-демо виджет** (правая часть на десктопе):
Небольшой терминал-интерфейс с анимированным typewriter-эффектом:
```
> Что умеет ИИ-агент для бизнеса?

  Отвечает клиентам в WhatsApp
  в 3 часа ночи без зарплаты.

  Обрабатывает входящие заявки
  и квалифицирует лиды за секунды.

  Ищет информацию, пишет черновики,
  суммирует встречи — без вашего участия.

  [Попробовать демо →]
```

**Фон hero**:
- Mesh gradient: три radial-gradient (violet #7c3aed 35%, cyan #06b6d4 25%, transparent)
- Animated: медленный drift, период 20 секунд
- Noise texture overlay: 4% opacity
- Полностью тёмный, контент читается без усилий

**Карточка 1 — Сайты**

Иконка: глобус или браузер (SVG, акцентный цвет)

**Заголовок**: «Сайт — ваш лучший продавец»

**Описание**:
> Создаю сайты, которые привлекают и конвертируют — не просто информируют. С нуля, под ваш бренд, с SEO-структурой и текстами, которые продают.

**Список (3 пункта)**:
- Превращает посетителей в заявки, а не просто показывает "кто мы"
- Запускается за 2–4 недели — без шаблонных компромиссов
- Адаптирован под мобильные, быстрый, оптимизирован под поиск

**CTA под карточкой**: «Примеры сайтов →»

**Карточка 2 — Приложения**

Иконка: мобильный телефон или окно приложения

**Заголовок**: «Приложение под ваши процессы»

**Описание**:
> Готовые инструменты заставляют вас подстраиваться под них. Кастомное приложение подстраивается под вас — и делает именно то, что нужно бизнесу.

**Список**:
- От идеи до рабочего продукта — без шестимесячных ожиданий
- Интерфейс, которым команда будет реально пользоваться
- Масштабируется вместе с ростом бизнеса

**CTA**: «Примеры приложений →»

---

**Карточка 3 — ИИ-агенты** ← помечена тегом «Новое» или «Горячо»

Иконка: нейросеть / искра / мозг

**Заголовок**: «ИИ-агент: сотрудник без зарплаты»

**Описание**:
> ИИ-агент — это не чат-бот на сайте. Это цифровой сотрудник, который обрабатывает задачи, отвечает клиентам, ищет данные и принимает решения — 24/7, без выходных.

**Список**:
- Отвечает на вопросы клиентов в любое время суток
- Обрабатывает входящие, квалифицирует лиды, пишет черновики

**Q: Сколько стоит?**

A: Зависит от проекта. Ориентиры:
- Лендинг / сайт-визитка: от 60 000 ₽, срок 2–3 недели
- Многостраничный сайт: от 120 000 ₽, срок 3–5 недель
- Веб-приложение: от 200 000 ₽, срок 4–8 недель
- ИИ-агент: от 80 000 ₽, срок 2–4 недели

## 6. Анимации и интерактив

### Принципы

1. **Анимация помогает читать** — никогда не мешает. Если убрать — смысл не теряется.
2. **Сначала CSS, потом JS** — нативные CSS scroll-driven animations для reveal-эффектов.
3. **Мобильные — без тяжёлых анимаций** — проверять `prefers-reduced-motion`.
4. **Скорость анимаций**: fast (150–200ms) для hover, medium (300–500ms) для reveals, slow (700ms+) для hero.

### Обязательные анимации

**Hero**:
```css
/* Анимация появления H1 при загрузке */
@keyframes heroIn {
  from { opacity: 0; transform: translateY(24px); }
  to   { opacity: 1; transform: translateY(0); }
}
.hero-title   { animation: heroIn 0.7s cubic-bezier(0.16, 1, 0.3, 1) both; }
.hero-sub     { animation: heroIn 0.7s 0.15s cubic-bezier(0.16, 1, 0.3, 1) both; }
.hero-buttons { animation: heroIn 0.7s 0.3s cubic-bezier(0.16, 1, 0.3, 1) both; }
.hero-stats   { animation: heroIn 0.7s 0.45s cubic-bezier(0.16, 1, 0.3, 1) both; }

/* Mesh gradient — медленный drift */
@keyframes meshDrift {
  0%, 100% { background-position: 0% 50%; }
  50%       { background-position: 100% 50%; }
}
.hero-bg { animation: meshDrift 20s ease infinite; }
```

**Typewriter в AI-виджете** (hero):

**CSS**:
- [ ] Tailwind v4 — только используемые стили в bundle

- [ ] Никаких runtime CSS-in-JS библиотек

**Сеть**:
- [ ] Vercel — auto Brotli compression, HTTP/3
- [ ] Cache headers: `Cache-Control: public, max-age=31536000, immutable` для статики
